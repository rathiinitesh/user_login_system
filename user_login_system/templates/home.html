This is a simple user login app with features like registeration, login, logout, change password, change profile pic etc. with an admin overseeing everything and having the ablity to delete and approve any user anytime.

Home Page:

This is the Home Page. So this is the overview/summary of this app. Focus, of this app is not on the design but on the backend (flask, python etc.) part of development.

About Page:

About Page contains "paginated" list of all the users.

Register Page:

Register Page is a form for new users to register using a unique email. Along with a link to go to login page if the user is already registered.

Login Page:

Login Page is also a form for user to login. It also contains a link to reset password in case the user forgot his/her password (an email is sent to the registered email id for the user to click and change password) and a link to register in case the user is not a registered user.

Admin Panel Page:

Admin Panel page is the page which only the admin can access but only after logging in. Here the admin can checkout non-verified users and can approve, disapprove and delete them. It also displays basic admin info as well.

User Dashboard Page:

User Dashboard shows all the info of a particular user and also contains a form for user to update his/her account info.

Other Pages:

Other Pages like for resetting password etc. only come into play when a user has their need to.
